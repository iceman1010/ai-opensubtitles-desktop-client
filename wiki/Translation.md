# Translation

Translate existing subtitle files between 100+ languages.

## Supported Input Formats

- **SRT** (SubRip Text)
- **VTT** (WebVTT)

Both formats support full Unicode for international characters.

## How to Translate Subtitles

### Step 1: Select File
Choose an existing subtitle file (SRT or VTT) by:
- Drag & drop onto the application window
- Or click to browse and select

### Step 2: Configure Languages
- **Source Language**: Set the original language or enable auto-detection
- **Target Language**: Choose from 100+ available languages
- **Translation Model**: Select the AI translation model

### Step 3: Process
Click "Start Translation" to begin.

### Step 4: Monitor
Track progress with live credit updates in the status bar.

### Step 5: Review & Save
- Preview the translated content
- Export translated subtitles to your desired location

Multiple translation models are available:

Check the "Info" section in the app for current model options and pricing.

## Auto-Detection

Enable automatic language detection when:
- You're unsure of the source language
- Working with multilingual content
- The source language varies within the file

## Tips for Best Results

1. **Original Quality**: Better original subtitles produce better translations
2. **Formatting**: VTT and SRT formatting is preserved in translation
3. **Special Characters**: Unicode support ensures international characters are handled correctly
4. **Context**: Some translation models consider context for better accuracy
