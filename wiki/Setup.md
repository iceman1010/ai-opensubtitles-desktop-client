# Setup Guide

## Getting API Credentials

Before using the application, you need to obtain API credentials from OpenSubtitles:

1. **Register an Account**: Visit [www.opensubtitles.com](https://www.opensubtitles.com) and create an account
2. **Get API Key**: Navigate to your account settings to generate an API key
3. **Purchase Credits**: The app uses a credit-based system for transcription and translation

> **Important**: Use Opensubtitles.com (not .org) when buying credits to ensure compatibility with this app.

## Configuring the Application

1. **Launch the Application**: Open the desktop client
2. **Enter Credentials**: On the login screen, enter:
   - Username
   - Password
   - API Key
3. **Save**: Your credentials will be securely stored for future sessions

## First-Time Setup Tips

### Language Preferences
- Set your preferred default language in Settings
- Enable auto-detection for unknown content

### Output Settings
- Choose default output format (SRT or VTT)
- Set default output directory

### Network
- The app automatically detects online/offline status
- Network errors are handled with automatic retry using exponential backoff

## Troubleshooting Login Issues

- **Invalid credentials**: Double-check your username, password, and API key
- **Account not verified**: Ensure your OpenSubtitles account is verified
- **No credits**: Purchase credits at ai.opensubtitles.com

## Updating the Application

The application includes an auto-updater that automatically checks for and installs updates. You can also manually check for updates in the app settings.
