# Batch Subtitle Processing

Save time and effort by processing multiple media and subtitle files simultaneously with the AI Opensubtitles Desktop Client. Our powerful batch processing feature includes efficient queue management, real-time progress tracking, and allows you to auto-generate subtitles or translate SRT files in bulk.

## Accessing Batch Mode

Navigate to the **"Batch"** tab within the application to access all bulk processing features.

## Adding Files to the Batch Queue

### Drag & Drop Interface
- Simply drag multiple files from your computer and drop them onto the Batch screen.
- You can mix and match file types: drop audio files, video files, and existing subtitle files (SRT/VTT) all at once.

### Standard File Selector
- Click to open the native file browser.
- Select multiple files simultaneously to add them to your processing queue.

## Configuring Batch Operations

### Global Settings (Apply to All)
Quickly set options that apply to every file in your current queue:
- **Target Language**: Set the output language for transcription or translation.
- **AI Model Selection**: Choose the AI model to balance speed and accuracy.
- **Output Format**: Choose to export all files as **SRT** or **VTT**.
- **Output Directory**: Define a single folder where all generated subtitles will be saved.

### Individual File Settings
You maintain granular control. Customize settings for individual files within the batch:
- Override the global language selection for specific files.
- Change the output format (SRT/VTT) on a per-file basis.

## Advanced Queue Management

### Reorder Files
Drag and drop items within the queue to prioritize the processing order.

### Remove Files
- Easily remove individual files from the queue if added by mistake.
- Use the "Clear" function to remove all files and start a fresh batch.

### Modify Individual Items
Click on any file currently in the queue to adjust its specific settings before processing begins.

## Executing the Batch Processing

### Starting the Batch Operation
Click **"Start Batch"** to begin processing all files sequentially in your queue.

### Real-Time Progress Tracking
- View an overall progress bar for the entire batch operation.
- See the individual processing status for each file.
- Monitor your API credit usage in real-time.

### Review and Completion
- Once finished, review all processed files within the app.
- Batch save all resulting subtitle files to your chosen output directory.

## Tips for Efficient Batch Processing

1. **Order by Priority**: Place your most important or urgent files at the top of the queue.
2. **Credit Management**: Ensure your account has sufficient API credits before initiating large batch operations.
3. **Network Stability**: A stable internet connection is highly recommended for processing large numbers of files without interruption.
4. **Output Organization**: Set a dedicated output folder to keep your newly generated batch subtitles neatly organized.
