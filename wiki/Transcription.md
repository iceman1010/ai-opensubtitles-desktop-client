# Transcription

Convert speech in audio and video files to text subtitles using AI-powered transcription.

## Supported Input Formats

### Video
MP4, MKV, AVI, MOV, WMV, WebM, FLV, 3GP, and 20+ more formats

### Audio
MP3, WAV, FLAC, AAC, OGG, M4A, WMA, AIFF, and 15+ more formats

## How to Transcribe a File

### Step 1: Select File
- Drag & drop an audio or video file onto the application window
- Or click to browse and select a file

### Step 2: Configure Options
- **Language**: Choose the spoken language or enable auto-detection
- **AI Model**: Select from multiple transcription models (quality/speed options)
- **Output Format**: Choose SRT or VTT
- **Return Content**: Enable/disable to receive the full transcription text

### Step 3: Process
Click "Start Transcription" to begin the process.

### Step 4: Monitor Progress
Watch the real-time progress in the status bar. The app displays:
- Current operation status
- Credit balance updates
- Processing progress

### Step 5: Review & Save
- Preview the generated subtitles with syntax highlighting
- Export to your desired location

## AI Models

The application supports multiple AI transcription models:
- Different models offer varying quality/speed trade-offs
- Check the "Info" section in the app for current model options and pricing

## Language Detection

The app can automatically detect the spoken language in your media file:
- Enable auto-detection when unsure of the source language
- Saves time for multi-language content

## Tips for Best Results

1. **Audio Quality**: Clear audio produces more accurate transcriptions
2. **Multiple Speakers**: The AI handles multiple speakers automatically
3. **Background Music**: Minimal background audio improves accuracy
4. **Accents**: Select the appropriate language variant if available
