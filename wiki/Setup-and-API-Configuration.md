# Setup and API Configuration

Configure your AI Opensubtitles Desktop Client to enable powerful auto subtitle generation and AI translation features. This guide walks you through obtaining your OpenSubtitles AI API credentials, adding them to the client, and completing the first-time setup for generating subtitles efficiently.

## Getting API Credentials

Before using the application for audio-to-text transcription or translating SRT files, you need to obtain API credentials from OpenSubtitles:

1. **Register an Account**: Visit [www.opensubtitles.com](https://www.opensubtitles.com) and create a free account.
2. **Get API Key**: Navigate to your account settings to generate a secure API key.
3. **Purchase Credits**: The app uses a credit-based system for high-quality transcription and translation services.

> **Important**: You must use Opensubtitles.com (not .org) when buying credits to ensure complete compatibility with this AI desktop app.

## Configuring the Application

1. **Launch the Application**: Open your newly installed AI Opensubtitles desktop client.
2. **Enter Credentials**: On the main login screen, enter your:
   - Username
   - Password
   - API Key
3. **Save**: Your credentials will be securely stored for future sessions, allowing you to seamlessly auto-generate subtitles.

## First-Time Setup Tips

### Language Preferences
- Set your preferred default language in the Settings menu.
- Enable auto-detection for working with multilingual or unknown audio content.

### Output Settings
- Choose your preferred default subtitle output format: **SRT (SubRip Text)** or **VTT (WebVTT)**.
- Set a default output directory to keep your generated subtitles organized.

### Network and Resiliency
- The app automatically detects online and offline status.
- Network errors during API calls are handled safely with an automatic retry system using exponential backoff, ensuring large transcription tasks complete successfully.

## Troubleshooting Login Issues

- **Invalid credentials**: Double-check your username, password, and API key for typos.
- **Account not verified**: Ensure your OpenSubtitles account has been verified via email.
- **No credits**: Purchase additional credits for AI processing at ai.opensubtitles.com.

## Auto-Updating the Application

The AI Opensubtitles client includes a built-in auto-updater that automatically checks for and installs the latest features and bug fixes. You can also manually check for updates directly in the app settings to ensure you always have the best subtitle generation tools.
