# Batch Processing

Process multiple files simultaneously with queue management and progress tracking.

## Accessing Batch Mode

Navigate to the "Batch" screen in the application to access batch processing features.

## Adding Files

### Drag & Drop
- Drag multiple files simultaneously onto the Batch screen
- Supports both audio/video files and subtitle files

### File Selector
- Click to open the file browser
- Select multiple files at once

## Configuring Batch Operations

### Global Settings
Set options that apply to all files in the queue:
- Target language
- AI model selection
- Output format (SRT/VTT)
- Output directory

### Individual File Settings
You can also customize settings for individual files:
- Override language selection
- Change output format per file

## Queue Management

### Reorder Files
Drag and drop to change the processing order

### Remove Files
- Remove individual files from the queue
- Clear all files to start fresh

### Modify Individual Items
Click on any file in the queue to adjust its specific settings

## Processing

### Starting Batch Operation
Click "Start Batch" to begin processing all files in the queue

### Progress Tracking
- View overall progress for the entire batch
- See individual file status
- Monitor credit usage in real-time

### Completion
- Review all processed files
- Batch save all results to your chosen directory

## Tips for Batch Processing

1. **Order by Priority**: Place most important files first
2. **Credit Management**: Ensure sufficient credits before starting large batches
3. **Network Stability**: A stable internet connection is recommended
4. **Output Organization**: Set a dedicated output folder for batch results
